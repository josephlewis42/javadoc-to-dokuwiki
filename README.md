# javadoc-to-dokuwiki
A javadoc to dokuwiki syntax converter. This is based on the excellent [Javadoc-to-Markdown converter by delight-im](https://github.com/delight-im/Javadoc-to-Markdown)



# Usage

`node javadoc-to-dokuwiki.js myjavafile`
